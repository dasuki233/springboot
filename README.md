# 勤工助学管理系统
## 项目简介
本系统分为三个角色：管理员、教师、学生

管理员功能：班级管理、考勤管理、留言管理、岗位管理、新闻公告管理、岗位工资管理、人员管理
教师功能：新闻公告、考勤管理、岗位管理、修改个人资料
学生功能：新闻公告、考勤情况、岗位申请、我的工资、在线留言

## 系统开发具体环境
- 开发环境：IntelliJ IDEA Ultimate
- 前端框架：[ElementUI](https://element.eleme.io/#/zh-CN)
- 数据库：MySQL 8.0.29
- Java:Java 1.8
- SpringBoot 2.6.3
- MyBatis-Plus 3.4.1
- Maven 3.6.3
